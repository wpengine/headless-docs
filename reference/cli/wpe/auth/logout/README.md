# wpe auth logout

## Description
Logout of the WP Engine Headless CLI

## Usage

```bash
$ wpe apps logout [OPTIONS]
```

## Extended Description

The `wpe auth logout` will revoke all permissions granted to the WP Engine Headless CLI and delete all stored credentials.
